## Description
This project analyzes Python as an open-source software, including its philosophy, Linux footprint, ecosystem, and comparison with proprietary alternatives.

## Scripts

### Script 1 - System Identity Report
Displays system details like kernel, user, uptime, and OS information.

### Script 2 - FOSS Package Inspector
Checks whether Python is installed and shows its details.

### Script 3 - Disk and Permission Auditor
Analyzes directory permissions and storage usage.

### Script 4 - Log File Analyzer
Counts occurrences of keywords in log files.

### Script 5 - Manifesto Generator
Creates a personalized open-source philosophy statement.
